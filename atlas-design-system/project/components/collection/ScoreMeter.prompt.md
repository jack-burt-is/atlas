The Outdoor Score + level meter — the signed-in user's headline status, like Gamerscore.

```jsx
<ScoreMeter score={14820} level={24} levelProgress={62} />
<ScoreMeter score={14820} level={24} levelProgress={62} toNext="680 pts to Level 25" />
```

Always gold-glowing. `levelProgress` is a percent toward the next level. Place it at the top of the Home dashboard and Profile.

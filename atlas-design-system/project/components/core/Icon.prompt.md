Lucide icon wrapper — use it anywhere Atlas needs an icon; the brand uses Lucide (2px stroke) exclusively.

```jsx
<Icon name="mountain" size={20} />
<Icon name="trophy" size={16} color="var(--gold-400)" />
```

Requires `<script src="https://unpkg.com/lucide@latest"></script>` on the page. Common Atlas icons: `mountain`, `trophy`, `map`, `map-pin`, `flag`, `route`, `compass`, `award`, `target`, `lock`, `check`, `chevron-right`, `footprints`, `tent`.
